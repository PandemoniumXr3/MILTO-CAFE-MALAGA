  # MILTO Specialty Coffee ☕

A modern café website concept for MILTO Specialty Coffee in Málaga.

Built with:
- HTML
- CSS
- Vanilla JavaScript

Focused on:
- premium café branding
- mobile-first UX
- specialty coffee atmosphere
- bilingual menu experience
- digital customer experience

---

## Features

- Responsive modern layout
- English / Spanish menu toggle
- Vegan / Vegetarian / Gluten-Free tags
- Smooth scroll reveal animations
- Interactive image hover effects
- Instagram integration
- Google Maps integration
- QR-based café experience
- Mobile-first design
- Clean café branding system

---

## QR Experience

The website includes a digital café system with QR codes for:

- Live menu
- Google reviews
- WiFi access
- Instagram

Designed to improve:
- customer convenience
- engagement
- reviews
- digital interaction inside the café
